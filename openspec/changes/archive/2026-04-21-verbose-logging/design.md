## Context

Текущая реализация использует pino для логирования с уровнем по умолчанию 'info'. Нужно добавить возможность расширенного логирования через CLI флаг.

## Goals / Non-Goals

**Goals:**
- Добавить флаг `-v, --verbose` в CLI
- При verbose использовать уровень логирования 'debug'
- Логировать все команды пользователей

**Non-Goals:**
- Изменение формата логов
- Логирование в файл (уже поддерживается)

## Decisions

### CLI флаг
Добавить `verbose: boolean` в интерфейс CLIArgs и обрабатывать через minimist.

### Logger модификация
Передавать verbose в createLogger и устанавливать уровень 'debug' если verbose = true.

### Telegraf middleware
Добавить middleware для логирования всех входящих команд в debug режиме.

## Risks / Trade-offs

- **Риск**: Слишком много логов при verbose → Mitigation: только команды, не все события
