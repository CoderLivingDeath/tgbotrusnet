## ADDED Requirements

### Requirement: Команда /start предоставляет приветствие и меню навигации
При команде /start неавторизованный пользователь должен получить приветственное сообщение с выбором категорий.

#### Scenario: Новый пользователь отправляет /start
- **WHEN** пользователь отправляет команду /start
- **THEN** бот отправляет приветственное сообщение с inline клавиатурой категорий

#### Scenario: Пользователь выбирает категорию
- **WHEN** пользователь нажимает кнопку категории
- **THEN** бот показывает список вопросов в этой категории или ответ если вопрос один

### Requirement: Категории отображаются корректно
Система должна показывать все доступные категории из базы данных.

#### Scenario: Отображение списка категорий
- **WHEN** загружаются категории из БД
- **THEN** каждая категория отображается как кнопка inline клавиатуры

### Requirement: Авторизация только для админ/оператор команд
Команды /admin и /operator требуют авторизации, но /start, /menu, /search - нет.

#### Scenario: Неавторизованный пользователь отправляет /admin
- **WHEN** пользователь отправляет /admin без авторизации
- **THEN** бот требует авторизацию

#### Scenario: Неавторизованный пользователь отправляет /start
- **WHEN** пользователь отправляет /start без авторизации
- **THEN** бот показывает приветствие с меню (НЕ требует авторизацию)
