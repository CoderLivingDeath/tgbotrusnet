## Why

В текущей реализации команды бота логируются базово, но нет возможности видеть подробный вывод всех команд для отладки. Пользователи и администраторы должны иметь возможность включить расширенное логирование для диагностики проблем.

## What Changes

- Добавить флаг `-v, --verbose` для CLI
- При указании флага выводить все входящие команды с аргументами
- Логировать callback_query данные
- Логировать сообщения пользователей

## Capabilities

### New Capabilities
- `verbose-logging`: Режим расширенного логирования всех команд

## Impact

- Изменения в `src/utils/cli.ts`: добавление --verbose флага
- Изменения в `src/index.ts`: передача verbose в logger
- Изменения в `src/services/logger.ts`: уровень логирования debug при verbose
